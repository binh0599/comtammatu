6 Agent Roles

Moi Agent so huu mot domain rieng biet. A1 la tiebreaker, A6 la gatekeeper. Cross-domain changes dung Handoff Protocol.

A1 — Architect (TIEBREAKER)

System Design, IA Authority. So huu URL structure, scope rules, route layout. Viet ADR cho moi quyet dinh lon. Review va approve moi schema change.

A2 — Frontend

UI/UX, Component System. Xay dung ScopeContextBar, route groups, @repo/ui. Dependencies: useScope() tu A4, TypeScript types tu A3.

A3 — Database

Schema, RLS, Migrations. brand_id migration, JWT claims hook, 11 module schemas, materialized views. Output: Generated TypeScript types.

A4 — Auth & RBAC

Permissions, Roles, Nav Config. nav-config.ts, ScopeProviders, has_role() function. Xoa 3 ban duplicate. Deliver useScope() truoc tuan 3.

A5 — Integration

External APIs, Edge Functions. PayOS, VietQR, E-invoicing, GrabFood, ShopeeFood, Zalo OA, ZNS. Tat ca secrets trong Vault.

A6 — CI/QA (GATEKEEPER)

Pipeline, Testing, Quality Gates. Fix 13 lint errors, branch protection, E2E Playwright, security scan. Khong code nao merge khi CI do.
